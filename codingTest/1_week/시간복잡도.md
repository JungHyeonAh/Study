# 시간복잡도

***

> 입력크기에 대해 어떤 알고리즘이 실행되는데 걸리는 시간이다.   
> **주요 로직의 반복횟수를 중점**으로 측정된다.   
   
```
ex)
console.time("test")

let a = 0   
for(let i=0; i<100; i++) {   
    a += i;   
}   
console.timeEnd("test")   
```

이와 같은 코드가 있을 때 console.time 코드를 사용해서 시간을 측정할 경우   
-> 컴퓨터 사양 등 여러 요소들로 인해 시간이 다 다르게 측정이 된다.   
**때문에 주요 로직이 몇번 반복되었는지를 기준으로 측정한다.**
